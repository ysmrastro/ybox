# サンプルのクイズアプリ

## これはサンプルのクイズアプリで静的なウェブサイトです。

## 使い方

1. `index.html` をブラウザで開きます。
2. クイズを開始します。
3. クイズに回答します。
4. 結果を確認します。

## クイズの内容

韓国語教室の生徒向けのクイズアプリです。
日本語の単語に対する韓国語の答えの組み合わせを当てるクイズです。
20問準備して10問ランダムで出題されます。

内容は事前に生成AIにて作成しYAMLファイルを作成してください。
